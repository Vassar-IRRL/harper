# Elbow print files

Printable parts for HARPER’s 1-DoF elbow (flexion/extension), including the Dynamixel drive, pulley pieces, and connectors into the upper arm and forearm.

## Naming convention

Filenames use `elb_{feature}.stl`.

| Token | Meaning |
|-------|---------|
| `elb_` | Elbow |
| `fam` | Forearm side of the elbow |
| `uarm` | Upper-arm side of the elbow |
| `conn` | Connector (into forearm or from upper arm) |
| `dxl` | Dynamixel shaft / spool / branch |
| `pulley` | Cable pulley path |
| `spacer` / `pin` | Alignment and spacing hardware |
| `tool` | Printable assembly / alignment aid (not a robot part) |
