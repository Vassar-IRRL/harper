# Shoulder print files

Printable parts for HARPER’s 3-DoF shoulder (flexion/extension, abduction/adduction, and rotation), plus the connector into the elbow.

## Naming convention

Filenames use `{shd}_{feature}.stl`.

| Token | Meaning |
|-------|---------|
| `shd_` | Shoulder |
| `fex` | Flexion / extension |
| `aad` | Abduction / adduction |
| `rot` | Rotation (internal / external) |
| `ubrack` | U-bracket |
| `elb_conn` | Connector from shoulder rotation into the elbow |
| `lower_screw` / `upper_screw` / `outer_collar` | Elbow-connector hardware pieces |
