# Forearm print files

Printable parts for HARPER’s forearm roll joint: housings, Dynamixel mount, and planetary gear train.

## Naming convention

Filenames use `fam_{feature}.stl`.

| Token | Meaning |
|-------|---------|
| `fam_` | Forearm |
| `dxl` | Dynamixel holder / drive interface |
| `conn` | Connector attachment (top / bottom) |
| `orbit` / `planet` | Planetary gear set |
| `servo_translate` | Gear that couples servo motion into the roll stage |
| `housing` / `bearing_lock` / `threaded_screw` | Structural and bearing-retention pieces |
| `gear_pegs` | Pegs for gear alignment / assembly |
