# HARPER Documentation

Docusaurus site for the [HARPER Project](https://github.com/Vassar-IRRL/harper). Published at [paudelsamip.com.np/harper](https://paudelsamip.com.np/harper/).

## Development

```bash
cd docs
npm install
npm start
```

The dev server runs at [http://localhost:3000/harper/](http://localhost:3000/harper/).

## Build

```bash
npm run build
npm run serve
```

## Adding docs

Add `.md` or `.mdx` files under `docs/docs/`. The sidebar is autogenerated from that folder structure.
